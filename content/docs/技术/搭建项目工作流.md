## 下载模型

下载hfd

```bash
wget https://hf-mirror.com/hfd/hfd.sh
chmod a+x hfd.sh
```

设置环境变量

```bash
export HF_ENDPOINT=https://hf-mirror.com
$env:HF_ENDPOINT = "https://hf-mirror.com"
```

下载模型

```bash
./hfd.sh <模型>  --hf_username <YOUR_HF_USERNAME> --hf_token <hf_***>
```

上传模型

```bash
rsync -avP <源路径> s09011@211:<目标路径>
```
