    Hugo

    1. 创建网站

      hugo new site <name>
cd <name>

    2. 安装主题

      git clone https://github.com/alex-shpak/hugo-book.git themes/hugo-book

    3. 修改配置文件hogo.toml

      baseURL = '网址'
languageCode = 'zh'
title = '站名'
theme = 'hugo-book'

[params]
      BookTheme = 'dark'

    4. 启动服务

      hugo server

    5. 将项目上传到Github仓库，使用Netlify托管

      git init
git add .
git commit -m <message>
git remote add origin <url>
git push -u origin main
