    健康 #

    运动、睡眠、饮食、节制。看似简单的事实则最难，如早睡，控制体重。

    时间 #

    一天的时间可以粗略地分成三份，即早上、下午、晚上，每份3-4小时。一份用于科研，包括读写改论文、写代码跑实验，保证科研的进展和毕业的要求；一份用于学习，包括课程、文档，保证能力的提升；一份用于机动，包括杂事、运动、娱乐，处理有截止日期的紧急任务，灵活安排。

    什么是浪费时间？娱乐和无所事事是浪费时间吗？不一定，如果你很享受这段时间且不后悔，那它就是非常有意义有价值的，真正的浪费时间是事后会后悔的事情。B站是把双刃剑，一方面B站有大量优质的学习资源，另一方面B站有大量有趣的视频吸引注意力。

    内卷 #

    努力和内卷的区别是什么？有人会将这两个词画等号，或者把内卷理解成努力Plus。但它们有本质的区别，努力是了实现自己的想要实现的目标，而内卷是随大流，顺应外界的期待。高考的独木桥也罢，难道人生只是一条单行道和无数所谓的关键节点吗？以读研为例，很多人并不真的志在学术与科研，但还是在环境的影响下走进这座围城。我认为没有人天生想躺平，无非是负反馈太多的一种自我保护机制，

    意义 #

    人生的意义是什么？人生没有意义，世界是草台班子，生老病死以外没有要紧事。我认为人最应该追求内心的平和、满足与自洽，想做的事不必在乎得失，想做就是最充分的理由。我们每个人都应该有自己的一杆秤（评价标准），任人评说都无法动摇，也没人能定义我们的成败。我们都是独立自主的人，PUA不足为惧，能动摇你的只有能威胁生存的东西和想要追求的东西。

    爱的反思 #

    面对PUA，感同身受才能认识到其可怕之处。我需要反思，只顾表达自己的感受，用痛苦博取同情是一种道德绑架，是以爱的名义伤害爱的人，但覆水难收。我曾经觉得只要可能性不为零，就能做成，现在看来有些唯心。
